API Quick Reference

POST /api/live/streams

Create a new live stream slot. Returns RTMP ingest URL.

POST /api/live/streams/{id}/start

Launch FFmpeg and begin HLS output for a live stream.

POST /api/live/streams/{id}/stop

Stop a live broadcast and finalize the recording.

GET /api/live/streams

List all registered live streams and their status.

POST /api/vod/videos

Upload a video file. Transcoding starts automatically.

GET /api/vod/videos/{id}

Get VOD details including transcoding progress (0–100%).

GET /api/vod/videos

List all VOD assets (uploaded, processing, ready, failed).

DELETE /api/vod/videos/{id}

Delete a VOD asset and all its transcoded files.
